This div has a shadow border!

    The box-shadow syntax used here is:

  1. The first value (0) is the horizontal offset of the shadow. Setting it to 0 keeps the shadow centered horizontally around the element.
  2. The second value (0) is the vertical offset. Similarly, setting it to 0 keeps it centered vertically.
  3. The third value (0) is the blur radius. Keeping it at 0 ensures the shadow remains solid, resembling a border.
  4. The fourth value (10px) is the spread radius, which essentially determines the thickness of the shadow. Adjusting this value will change the thickness of your "border."
  5. The color value (#8a2be2) defines the color of the shadow, which in this case acts as the border color.
